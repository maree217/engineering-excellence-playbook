# Assessment Tools

**Structured frameworks for evaluating client capabilities and planning transformations.**

---

## Available Assessments

### [AI Maturity Assessment](ai-maturity-assessment.md)
Comprehensive evaluation of AI readiness across 5 dimensions:
- Strategy & Leadership
- Data Foundation
- Technology & Infrastructure
- Skills & Organization
- Governance & Ethics

**Duration**: 4-hour workshop | **Output**: Scorecard + 12-month roadmap

---

### [Cloud Readiness Assessment](cloud-readiness-assessment.md)
Evaluate readiness for cloud adoption or migration:
- Current infrastructure inventory
- Application portfolio assessment
- Security and compliance requirements
- Skills and organizational readiness
- Cost analysis and TCO projection

**Duration**: 2-3 days | **Output**: Migration readiness report + prioritized application list

---

### [Technical Debt Audit](technical-debt-audit.md)
Identify, quantify, and prioritize technical debt:
- Code quality metrics
- Architecture debt mapping
- Dependency analysis
- Security vulnerability assessment
- Remediation cost estimation

**Duration**: 1-2 weeks | **Output**: Debt register + prioritized remediation plan

---

### [Well-Architected Review](well-architected-review.md)
Platform-agnostic architecture review based on 5 pillars:
1. **Reliability**: Fault tolerance, disaster recovery, self-healing
2. **Security**: Identity, encryption, network security, compliance
3. **Cost Optimization**: Resource efficiency, reserved capacity, FinOps
4. **Operational Excellence**: Monitoring, automation, runbooks
5. **Performance**: Scalability, latency, throughput optimization

**Duration**: 1 week | **Output**: Findings report + improvement roadmap

## How to Conduct Assessments

### Pre-Workshop Preparation
1. Send pre-assessment questionnaire (1-2 weeks before)
2. Request documentation (architecture diagrams, data catalogs, org charts)
3. Identify key stakeholders (technical leads, business owners, executives)
4. Prepare assessment materials and scoring templates

### Workshop Facilitation
1. Set clear objectives and ground rules
2. Use structured questions to gather evidence
3. Score dimensions collaboratively with stakeholders
4. Document specific examples for each score
5. Identify quick wins during discussion

### Post-Workshop Deliverables
1. Compile scores into visual scorecard
2. Write narrative assessment report
3. Develop prioritized roadmap
4. Create executive summary presentation
5. Schedule follow-up to present findings

## Assessment Output Templates

All assessments include:
- **Scorecard**: Visual representation of current state
- **Gap Analysis**: Comparison to target state
- **Roadmap**: Prioritized action plan with milestones
- **Business Case**: ROI projections for recommended investments
- **Executive Summary**: 2-page overview for leadership
