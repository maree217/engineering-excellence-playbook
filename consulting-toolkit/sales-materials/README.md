## How to Use These Materials

### Initial Discovery Call (30 minutes)
1. Use **Competitive Positioning** to address "why not big consulting" questions
2. Share industry-relevant case study from `../case-studies/`
3. Demonstrate **ROI Calculator** with preliminary estimates

### Strategy Workshop (4 hours)
1. Send **Strategy Workshop Agenda** as pre-read
2. Collect pre-workshop assessment responses
3. Facilitate using the structured agenda
4. Output: Prioritized roadmap and pilot proposal

### Post-Workshop Follow-up
1. Generate proposal using templates from `../proposal-templates/`
2. Include ROI calculations from workshop
3. Reference relevant case studies

### vs Big Consulting
- **Speed**: 2-8 weeks vs 12-24 months

- **Transfer**: Build internal capabilities vs dependencies
